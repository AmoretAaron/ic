The PayloadBuilder is responsible for creating and validating payload that is included in consensus blocks.

) -> Result<BatchPayload, XNetPayloadError>

Produces a payload that is valid given past_payloads and context.

past_payloads contains the Payloads from all blocks above the certified height provided in context, in descending block height order.

It returns a BatchPayload if the payload building is success, or an error if it fails (which at the moment is only a transient error from building XNet payload, but will be extended in the future).
