ic_types

[−][src]Module ic_types::artifact

Artifact related types.

Notably it includes the following definitions and their sub-types:

  • Artifact
  • ArtifactTag
  • ArtifactId
  • ArtifactAttribute
  • ArtifactFilter

An ArtifactKind trait is provided for convenience to carry multiple type definitions that belong to the same "artifact kind".

All Artifact sub-types must also implement [ChunkableArtifact] trait defined in the chunkable module.
