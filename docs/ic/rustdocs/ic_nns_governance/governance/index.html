ic_nns_governance

[−][src]Module ic_nns_governance::governance

The 'governance' module contains the canister (smart contract) that manages neurons, proposals, voting, voter following, voting rewards, and the code necessary to execute accepted proposals.

The governance canister interfaces with the 'ledger' canister to deal with the transfer of ICP (Internet Computer Protocol) tokens between ledger accounts, and with the 'registry' canister to distribute configuration information to all nodes of all subnetworks that participate in the Internet Computer (IC).
