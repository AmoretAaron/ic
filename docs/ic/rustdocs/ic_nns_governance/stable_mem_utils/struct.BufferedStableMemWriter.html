pub struct BufferedStableMemWriter { /* fields omitted */ }

An implementation of BufMut that writes to stable memory in chunks. Chunk size is specified on initialization, in BufferedStableMemWriter::new.

Note that you need to drop this, or call flush(), after using the BufMut methods, to write the buffer contents into the stable memory.
