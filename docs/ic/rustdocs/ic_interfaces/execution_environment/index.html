An underlying struct/helper for implementing select() on multiple AsyncResult's. If an AsyncResult is really an ongoing computation, we have to obtain its result from a channel. However, some AsyncResults are of type EarlyResult, which only emulates being async, but in reality is a ready value (mostly used for early errors). In such case, there is no channel present and we can simply return the value without waiting.
